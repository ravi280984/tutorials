# cloud/azure

Purpose

Azure tutorials and samples: deploying models to Azure ML, using Blob Storage, and examples of integrating Azure services with AI workloads.

Prerequisites

- Azure account and Azure CLI installed
- Python 3.10+

Run instructions

- Configure CLI: `az login`
- Follow individual examples for deploying to Azure ML, using AKS, or serverless options.

Notes

- Use managed identities and service principals for automation.
- Monitor cost and remove resources after use.
