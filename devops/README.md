# devops

Purpose

DevOps folder holds examples for CI/CD, infrastructure automation, and GitOps patterns tailored for AI projects.

Prerequisites

- Git
- Docker (optional)
- Familiarity with GitHub Actions or other CI systems

Run instructions

- Read subfolders for specific workflows; to run CI locally use tools like `act` or run tests with `pytest`.

Notes

- Keep workflows modular and reusable. Use composite actions or reusable workflows when possible.
- Secrets should be stored in GitHub secrets or a secure vault.
