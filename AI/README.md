# AI

Purpose

This folder contains AI-focused tutorials and experiments: agent patterns (MCP), RAG examples, model evaluation, and small demos that illustrate end-to-end workflows for research and prototyping.

Prerequisites

- Python 3.10+
- Install repository dependencies: `pip install -r requirements.txt`
- Optional: Docker for containerized runs

Run instructions

- Explore subfolders (`mcp/`, `rag/`) for focused examples with their own READMEs.
- Typical steps:
  1. Create and activate a virtual environment
  2. pip install -r requirements.txt
  3. Run the example scripts or notebooks in the subfolders

Notes

- Keep large datasets and model weights out of the repo; use external storage referenced in `.gitignore`.
- Use environment variables for API keys and credentials.

Guidelines

- Follow best practices for reproducibility: pin package versions, include small sample datasets for demos, and add clear instructions for reproducing results.
