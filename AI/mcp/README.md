# AI/mcp

Purpose

This folder contains examples and starter code for Model Context Protocol (MCP) style agents and local agent orchestration. Use these resources to learn how to build agents that maintain state and context across interactions.

Prerequisites

- Python 3.10+
- Install dependencies: `pip install -r requirements.txt`

Run instructions

- Follow individual notebook or script README files in this folder for step-by-step examples.
- Typical workflow:
  1. Activate environment
  2. pip install -r ../../requirements.txt
  3. Run example: `python example_agent.py` (if present)

Notes

- Keep model weights and large datasets outside the repository (see `.gitignore`).
- Use environment variables for credentials.
